## 1. PHQ-9 - 抑郁症筛查量表

**来源**: Kroenke K, Spitzer RL, Williams JB. The PHQ-9: validity of a brief depression severity measure. J Gen Intern Med. 2001;16(9):606-613.

**测评维度**: 过去两周的抑郁症状

### 指导语
过去两周，您有多少时间受到以下问题困扰？

### 选项（统一4级）
| 分值 | 选项 |
|------|------|
| 0 | 完全不会 |
| 1 | 好几天 |
| 2 | 一半以上的天数 |
| 3 | 几乎每天 |

### 题目
1. 做事时提不起劲或没有兴趣
2. 感到心情低落、沮丧或绝望
3. 入睡困难、睡不安稳或睡眠过多
4. 感觉疲倦或没有活力
5. 食欲不振或吃太多
6. 觉得自己很糟——或觉得自己很失败，或让自己或家人失望
7. 对事物专注有困难，例如阅读报纸或看电视时
8. 动作或说话速度缓慢到别人已经觉察？或正好相反——变得比平日更烦躁或坐立不安，动来动去
9. 有不如死掉或用某种方式伤害自己的念头

### 计分与阈值
总分为所有题目分值之和，范围 0-27。

| 分数范围 | 严重程度 | 颜色编码 | 得分标签 |
|----------|----------|----------|----------|
| 0-4 | 无抑郁症状 | #22C55E (绿) | 正常范围 |
| 5-9 | 轻度抑郁 | #EAB308 (黄) | 轻度 |
| 10-14 | 中度抑郁 | #F97316 (橙) | 中度 |
| 15-19 | 中重度抑郁 | #EF4444 (红) | 中重度 |
| 20-27 | 重度抑郁 | #DC2626 (深红) | 重度 |

### 解读文本模板

**0-4分：正常范围**
您的PHQ-9得分为{score}分，处于正常范围。这表明您在过去两周内没有明显的抑郁症状。请继续保持健康的生活方式，包括规律作息、适度运动和良好的社交互动。

**5-9分：轻度抑郁**
您的PHQ-9得分为{score}分，提示存在轻度抑郁症状。您可能在情绪、精力或兴趣方面有些困扰，但尚未严重影响日常生活。建议：增加户外活动、保证充足睡眠、与亲友交流感受。如症状持续两周以上，建议寻求专业心理咨询。

**10-14分：中度抑郁**
您的PHQ-9得分为{score}分，提示存在中度抑郁症状。这可能已经对您的日常生活、工作或人际关系产生了一定影响。强烈建议您预约心理咨询师或精神科医生进行专业评估。同时，请尝试保持规律生活，不要独自承受。

**15-19分：中重度抑郁**
您的PHQ-9得分为{score}分，显示存在中重度抑郁症状。抑郁症状正在显著影响您的正常生活功能。请尽快寻求专业帮助——预约精神科医生或心理治疗师。如果您有自伤念头，请立即拨打心理援助热线。

**20-27分：重度抑郁**
您的PHQ-9得分为{score}分，显示存在重度抑郁症状。这是需要立即关注的严重情况。请立即联系精神科医生或前往医院。如果您正在经历自伤或自杀念头，请马上拨打心理援助热线：希望24热线 400-161-9995，或拨打110/120。

**注意事项**
- 第9题（自伤念头）任何非0回答都需特别关注，无论总分多少
- PHQ-9是筛查工具，不是诊断工具，最终诊断须由专业医生完成

## 2. GAD-7 - 广泛性焦虑障碍量表

**来源**: Spitzer RL, Kroenke K, Williams JBW, Löwe B. A brief measure for assessing generalized anxiety disorder: the GAD-7. Arch Intern Med. 2006;166(10):1092-1097.

**测评维度**: 过去两周的焦虑症状

### 指导语
过去两周，您有多少时间受到以下问题困扰？

### 选项（同PHQ-9）
| 分值 | 选项 |
|------|------|
| 0 | 完全不会 |
| 1 | 好几天 |
| 2 | 一半以上的天数 |
| 3 | 几乎每天 |

### 题目
1. 感觉紧张、焦虑或急切
2. 无法停止或控制担忧
3. 对各种各样的事情担忧过多
4. 很难放松下来
5. 由于不安而无法静坐
6. 变得容易烦恼或急躁
7. 感到似乎将有可怕的事情发生而害怕

### 计分与阈值
总分范围 0-21。

| 分数范围 | 严重程度 | 颜色编码 | 得分标签 |
|----------|----------|----------|----------|
| 0-4 | 无焦虑症状 | #22C55E (绿) | 正常范围 |
| 5-9 | 轻度焦虑 | #EAB308 (黄) | 轻度 |
| 10-14 | 中度焦虑 | #F97316 (橙) | 中度 |
| 15-21 | 重度焦虑 | #EF4444 (红) | 重度 |

### 解读文本模板

**0-4分：正常范围**
您的GAD-7得分为{score}分，焦虑水平在正常范围内。这表明您在过去两周内没有明显的焦虑困扰。适度的焦虑是人体的正常应激反应，关键是与它和平共处。

**5-9分：轻度焦虑**
您的GAD-7得分为{score}分，提示存在轻度焦虑症状。您可能经常感到紧张或担忧，但总体上还能应对。建议练习正念冥想、深呼吸放松技巧，减少咖啡因摄入。如症状持续，可寻求心理咨询。

**10-14分：中度焦虑**
您的GAD-7得分为{score}分，提示存在中度焦虑症状。焦虑可能已经干扰到您的睡眠、工作或人际关系。强烈建议寻求专业帮助——认知行为疗法（CBT）对焦虑症状效果显著。

**15-21分：重度焦虑**
您的GAD-7得分为{score}分，显示存在重度焦虑症状。焦虑正在严重影响您的生活质量。请尽快预约精神科医生进行专业评估和治疗。在等待就医期间，请尽量避免独自承受，与信任的人谈谈您的感受。

## 3. PSS-10 - 压力感知量表

**来源**: Cohen S, Kamarck T, Mermelstein R. A global measure of perceived stress. J Health Soc Behav. 1983;24(4):385-396.

**测评维度**: 过去一个月的压力感知水平

### 指导语
在过去一个月里，您有多少时间会感到……

### 选项
| 分值 | 选项 |
|------|------|
| 0 | 从不 |
| 1 | 偶尔 |
| 2 | 有时 |
| 3 | 时常 |
| 4 | 总是 |

### 题目
1. 因发生意外的事情而感到心烦意乱
2. 感觉无法控制自己生活中重要的事情
3. 感到紧张不安和压力
4. 成功地处理了生活中的麻烦事（反向计分）
5. 感到自己能有效地应对生活中的重要变化（反向计分）
6. 对自己处理个人问题的能力感到有信心（反向计分）
7. 感到事情顺心如意（反向计分）
8. 发现自己无法处理必须做的事情
9. 有办法控制生活中的烦恼（反向计分）
10. 觉得自己是能掌控事情的（反向计分）

> **反向计分题**: 第4、5、6、7、9、10题，分值反转（0→4, 1→3, 2→2, 3→1, 4→0）

### 计分与阈值
总分范围 0-40。

| 分数范围 | 严重程度 | 颜色编码 | 得分标签 |
|----------|----------|----------|----------|
| 0-13 | 低压力感知 | #22C55E (绿) | 低压力 |
| 14-26 | 中等压力感知 | #EAB308 (黄) | 中等压力 |
| 27-40 | 高压力感知 | #EF4444 (红) | 高压力 |

### 解读文本模板

**0-13分：低压力感知**
您的PSS-10得分为{score}分，压力感知处于较低水平。这说明您目前能够较好地应对生活中的各种挑战，拥有良好的压力管理能力和心理弹性。

**14-26分：中等压力感知**
您的PSS-10得分为{score}分，压力感知处于中等水平。您正在经历一定程度的压力，这可能影响到您的情绪和精力。建议：定期运动、保证睡眠、学习时间管理技巧、建立健康的边界。

**27-40分：高压力感知**
您的PSS-10得分为{score}分，压力感知处于较高水平。您正承受着较重的压力负担，这可能对身心健康产生不利影响。强烈建议：识别压力源并尝试减少不必要的压力因素，学习放松技巧，必要时寻求心理咨询帮助。

## 4. RSES - Rosenberg自尊量表

**来源**: Rosenberg M. Society and the adolescent self-image. Princeton University Press; 1965.

**测评维度**: 整体自尊水平（自我价值感）

### 指导语
请根据您的真实感受，选择最符合的选项。

### 选项
| 分值 | 选项 |
|------|------|
| 1 | 完全不同意 |
| 2 | 不同意 |
| 3 | 同意 |
| 4 | 完全同意 |

### 题目
1. 总的来说，我对自己是满意的
2. 有时，我觉得自己一点都不好（反向计分）
3. 我觉得自己有一些优点
4. 我能把事情做得和大多数人一样好
5. 我觉得自己没有什么值得骄傲的（反向计分）
6. 有时我的确感到自己很没用（反向计分）
7. 我觉得自己是一个有价值的人，至少和别人在同一个水平上
8. 我希望能更看得起自己一些（反向计分）
9. 总的来说，我倾向于觉得自己是一个失败者（反向计分）
10. 我对自己持积极的态度

> **反向计分题**: 第2、5、6、8、9题，分值反转（1→4, 2→3, 3→2, 4→1）

### 计分与阈值
总分范围 10-40。

| 分数范围 | 严重程度 | 颜色编码 | 得分标签 |
|----------|----------|----------|----------|
| 10-17 | 低自尊 | #EF4444 (红) | 低自尊 |
| 18-24 | 中等偏低 | #F97316 (橙) | 中等偏低 |
| 25-31 | 中等 | #EAB308 (黄) | 中等自尊 |
| 32-40 | 高自尊 | #22C55E (绿) | 高自尊 |

### 解读文本模板

**10-17分：低自尊**
您的RSES得分为{score}分，处于低自尊水平。您可能经常自我怀疑，难以认可自己的价值。低自尊往往与抑郁和焦虑相关。建议通过认知行为疗法（CBT）改善自我评价，学会挑战负面的自我认知。寻求专业心理咨询会有很大帮助。

**18-24分：中等偏低**
您的RSES得分为{score}分，处于中等偏低水平。您有时会对自己不太满意，但并非全盘否定。建议练习自我肯定、记录每日成就、设定可实现的小目标来逐步建立自信。

**25-31分：中等自尊**
您的RSES得分为{score}分，处于中等自尊水平。您对自己有基本的认可，但在某些方面可能仍有疑虑。这是大多数人的正常范围，建议继续保持积极的自我对话。

**32-40分：高自尊**
您的RSES得分为{score}分，处于高自尊水平。您对自己持有积极的评价，拥有健康的自我价值感。高自尊与更好的心理健康和人际关系相关。请继续保持！

## 5. BFI-10 - 大五人格量表（简版）

**来源**: Rammstedt B, John OP. Measuring personality in one minute or less: A 10-item short version of the Big Five Inventory in English and German. J Res Pers. 2007;41(1):203-212.

**测评维度**: 人格五大维度（每维度2题，共10题）

### 指导语
以下是一些关于您自己的描述。请指出每个描述在多大程度上符合您的情况。

### 选项
| 分值 | 选项 |
|------|------|
| 1 | 完全不同意 |
| 2 | 有点不同意 |
| 3 | 中立 |
| 4 | 有点同意 |
| 5 | 完全同意 |

### 题目与维度对应

| 题号 | 题目 | 所属维度 | 计分方向 |
|------|------|----------|----------|
| 1 | 我认为自己是外向、善于交际的 | 外向性(E) | 正向 |
| 2 | 我认为自己是挑剔、爱争论的 | 宜人性(A) | 反向 |
| 3 | 我认为自己是可靠、自律的 | 尽责性(C) | 正向 |
| 4 | 我认为自己是紧张、容易心烦的 | 神经质(N) | 正向 |
| 5 | 我认为自己是有丰富想象力的 | 开放性(O) | 正向 |
| 6 | 我认为自己是内敛、安静的 | 外向性(E) | 反向 |
| 7 | 我认为自己是体谅、友善的 | 宜人性(A) | 正向 |
| 8 | 我认为自己是容易分心、懒散的 | 尽责性(C) | 反向 |
| 9 | 我认为自己是情绪稳定、不易心烦的 | 神经质(N) | 反向 |
| 10 | 我认为自己是对艺术不感兴趣的 | 开放性(O) | 反向 |

### 维度计分（每维度2题取均值，范围1-5）

| 人格维度 | 高分特征 | 低分特征 |
|----------|----------|----------|
| 外向性 Extraversion | 善于社交、精力充沛、热情 | 内敛、安静、独处偏好 |
| 宜人性 Agreeableness | 信任他人、合作、有同情心 | 竞争性、怀疑、自我中心 |
| 尽责性 Conscientiousness | 自律、有条理、可靠 | 随性、杂乱、拖延 |
| 神经质 Neuroticism | 焦虑、情绪波动、易紧张 | 情绪稳定、从容、抗压力强 |
| 开放性 Openness | 好奇、创造性、喜欢新体验 | 务实、传统、习惯熟悉的 |

### 解读文本模板

**外向性 (Extraversion): {score}/5**
{score >= 4 ? "高外向性：您喜欢社交互动，在人群中感到精力充沛，善于表达自己。" :
 score >= 2.5 ? "中等外向性：您在社交和独处之间能找到平衡，既能享受社交也能从独处中恢复精力。" :
 "低外向性（内倾）：您偏好独处或小圈子交往，在安静环境中更舒适，思考深入。"}

**宜人性 (Agreeableness): {score}/5**
{score >= 4 ? "高宜人性：您善于合作、富有同情心，倾向于信任他人，注重人际关系和谐。" :
 score >= 2.5 ? "中等宜人性：您在坚持原则和照顾他人感受之间能够平衡。" :
 "低宜人性：您倾向于独立思考、坚持自己的立场，不轻易妥协，具有批判性思维。"}

**尽责性 (Conscientiousness): {score}/5**
{score >= 4 ? "高尽责性：您自律性强、做事有条理、可靠负责。这种特质与学业和职业成功高度相关。" :
 score >= 2.5 ? "中等尽责性：您能在计划和灵活性之间找到平衡。" :
 "低尽责性：您喜欢随性而为，有较好的灵活性和创造力，但可能在计划和执行上需要更多努力。"}

**神经质 (Neuroticism): {score}/5**
{score >= 4 ? "高神经质：您容易感到紧张和焦虑，情绪波动较大。建议学习情绪调节技巧如正念冥想。" :
 score >= 2.5 ? "中等神经质：您在多数时候情绪稳定，但在压力情境下可能会有情绪波动。" :
 "低神经质（情绪稳定）：您情绪稳定、抗压能力强，能够从容应对生活中的挑战。"}

**开放性 (Openness): {score}/5**
{score >= 4 ? "高开放性：您喜欢新事物、富有想象力和创造力，对艺术和抽象概念感兴趣。" :
 score >= 2.5 ? "中等开放性：您对新体验持开放态度，但也能欣赏熟悉和传统的事物。" :
 "低开放性：您偏好熟悉和可预测的环境，务实且注重常识，喜欢具体的而非抽象的事物。"}
