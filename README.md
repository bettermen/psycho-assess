# 🧠 Psycho Assess - 专业心理测评

基于国际标准心理量表的专业心理测评工具，支持交互式答题与可视化HTML评估报告。

## 量表清单

| 量表 | 题目数 | 评估维度 | 来源 |
|------|--------|----------|------|
| PHQ-9 | 9题 | 抑郁症状筛查 | Kroenke et al., 2001 |
| GAD-7 | 7题 | 焦虑症状筛查 | Spitzer et al., 2006 |
| PSS-10 | 10题 | 压力感知评估 | Cohen et al., 1983 |
| RSES | 10题 | 自尊水平评估 | Rosenberg, 1965 |
| BFI-10 | 10题 | 大五人格画像 | Rammstedt & John, 2007 |

## 功能特性

- ✅ 交互式逐量表答题
- ✅ 自动反向计分 + 临床阈值对照
- ✅ PHQ-9 第9题自伤念头高危标记
- ✅ 可视化HTML报告（得分圆环/进度条/SVG雷达图）
- ✅ 红/橙/黄三级预警分级
- ✅ 个性化综合建议
- ✅ 心理援助热线

## 安装

```bash
# 复制到 WorkBuddy skills 目录
cp -r psycho-assess ~/.workbuddy/skills/
```
